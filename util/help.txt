Usage: run target.ext ...       run target with rule ext
       run ext target ...       run target with rule ext
       run rule run             run rule with empty target

Language rules: cpp         run C++, extra arguments are passed to g++
                java        run Java, extra arguments are passed to javac
                py          run Python, extra arguments are passed to python

Special rules: run help         print help file
               run clean        delete all compiled objects
               run clean dir    delete all compiled objects in dir
